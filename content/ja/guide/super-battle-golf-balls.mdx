ボールを管理するための基本的な操作は単純ですが、プレッシャーの中でマスターするのは困難です。地形、風、そしてあなたの顔面にロケットランチャーを狙っているかもしれない他のプレイヤーとの距離を考慮する必要があります。

| 操作 | アクション | プロのヒント |
| :--- | :--- | :--- |
| **左クリック（長押し）** | パワーチャージ | 常に100%で打つのは避けましょう。オーバーシュートするとウォーターハザードに捕まります。 |
| **スクロールホイール** | 角度調整 | 最大飛距離を出すには、通常45度の角度が最適です。 |
| **1, 2, 3, 4 キー** | クイック角度プリセット | 高速レース中に、よく使う角度へ瞬時に切り替えるために使用します。 |
| **スペースキー** | ジャンプ（アイテム使用時） | タイミングよくジャンプすることで、ボールが小さな障害物を飛び越えるのを助けます。 |

<Callout type="tip" title="角度の最適化">
ラフや木の後ろにスタックした場合は、スクロールホイールを使ってロフト（打ち出し角）を上げましょう。角度を高くすると、前方向への総飛距離は犠牲になりますが、垂直方向の障害物をクリアすることができます。
</Callout>

<Section icon={Target} title="ステップ・バイ・ステップ：完璧なショットを打つ方法">

スイングの仕組みをマスターすることは、打数を低く抑えるために不可欠です。あなたの**スーパーバトルゴルフボール**を常にフェアウェイに乗せるために、以下のステップに従ってください。

<Steps>
  <Step num="1" title="地形を調査する">
    ウォーターハザード、木、地雷がないか確認します。ルートがクリアな場合はフェアウェイを狙います。障害物がある場合は、より高いロフト角（打ち出し角）を使用する準備をしてください。
  </Step>
  <Step num="2" title="角度を調整する">
    スクロールホイールを使用して弾道を決定します。長距離の場合は45度が標準です。グリーンへのアプローチ（チップイン）では、ボールがカップを飛び越えて跳ねないように、低い角度（10〜20度）を使用します。
  </Step>
  <Step num="3" title="チャージしてリリース">
    左クリックを長押ししてパワーメーターを溜めます。メーターを注意深く観察してください。チャージのしすぎは、競技プレイにおいてペナルティ打数を課される主な原因となります。
  </Step>
  <Step num="4" title="移動手段で追撃">
    ボールが着地したら、すぐにその場所へ移動します。ゴルフカートやスプリングブーツを持っている場合は、対戦相手が立て直す前にここで使用して距離を縮めましょう。
  </Step>
</Steps>

<Callout type="info" title="ペナルティ打数">
OB（アウトオブバウンズ）や深い池に入ると、ペナルティ打数が課されます。ボールは前の位置にリセットされ、ラウンドタイマーの貴重な時間を失うことになります。
</Callout>

</Section>

<Section icon={Trophy} title="スコアとマッチの目標">

スーパーバトルゴルフの目標は2つあります。打数を最も少なく抑えることと、できるだけ早くカップに到達することです。従来のゴルフは打数のみを気にしますが、この「バトル」要素においては、遅いプレイヤーに不利益を与えるタイマーが導入されています。

| 打数 | 用語 | スコアへの影響 |
| :--- | :--- | :--- |
| **-3** | アルバトロス | トータルポイントへの破格のブースト。 |
| **-2** | イーグル | 大きなアドバンテージ。 |
| **-1** | バーディ | 優れたパフォーマンス。 |
| **0** | パー | 標準的な期待される結果。 |
| **+1** | ボギー | 軽微な後退。 |
| **DNF** | リタイア（未完走） | 最大ペナルティ。0ポイント。 |
