<Section icon={MousePointer2} title="Super Battle Golfの基本操作をマスターする">

**Super Battle Golfの操作**を理解することは、リーダーボードを支配するための第一歩です。従来のゴルフシミュレーターとは異なり、このゲームではボールの軌道とともにキャラクターの物理的な動きを管理する必要があります。インターフェースは直感的に設計されていますが、「メイヘム（大混乱）」要素があるため、精密なパットから防御的なスイングへと瞬時に切り替えられる必要があります。

**ビデオの見どころ:**
- **エネルギー管理**: 破壊的なホーミングショットのためにエネルギーをチャージする方法。
- **地雷の操作**: 爆発物を安全に解除するために必要な特定のボタンコンボ。
- **戦闘バフ**: 他のプレイヤーを攻撃することで得られる移動速度の上昇の実演。
- **環境プレイ**: 軌道レーザーやゴルフカートを使用してマップを移動する方法。

戦闘は主に「牽制」と「とどめ」の2つの状態に分かれます。単純なノーパワーヒットは、相手を転倒させるのに十分であり、重要なパットを阻止するのに最適です。しかし、相手がダウンした後は、最大パワーヒットをチャージしてエリアから一掃するのが最善の方法です。

| 戦闘ムーブ | 入力方法 | 最適な使用場面 |
| :--- | :--- | :--- |
| **クイックジャブ** | クリックを連打 | 相手がボールを打とうとしている時 |
| **ヘビースマッシュ** | 長押しして離す | 相手がすでにスタンしている、または崖の近くにいる時 |
| **ホーミングストライク** | 最大エネルギー + クリック | 素早く動くターゲットや特定のアイテムに当てたい時 |
| **ディフェンシブスピン** | エリア攻撃 | 複数の敵対プレイヤーに囲まれている時 |

<Callout type="warning" title="注意">
戦いに夢中になりすぎないでください。ポイントが得られるのは上位3位までに入った場合のみです。すでに3人がカップインしているなら、戦いをやめてアイテム収集や自分のラウンドの終了に集中しましょう。
</Callout>

<Section icon={Target} title="高度なショットタイプ：パッティングとチッピング">

精密さは**Super Battle Golfの操作**の第2の柱です。周囲で戦闘の混乱が起きていても、スコアを最大化するにはできるだけ少ないスイング数でボールをカップに入れる必要があります。ゲームでは、物理的なポジショニングとパワーレベルによって「パッティング」と「チッピング」を使い分けます。

<Steps>
  <Step num="1" title="ショットタイプの選択">
    「パット・オーバー」が必要か「チップ・アンダー」が必要かを判断します。パッティングは平坦な場所や短距離用で、チッピングは小さな障害物や荒れた地形を越えるのに役立ちます。
  </Step>
  <Step num="2" title="軌道の調整">
    カメラを使用してショットのラインを合わせます。ボールは環境オブジェクトや他のプレイヤーに跳ね返る可能性があることを忘れないでください。
  </Step>
  <Step num="3" title="エネルギーのチャージ">
    スイングボタンを長押ししてパワーを溜めます。エネルギーが最大になるとレティクルが変化し、**ホーミングショット**が使用可能になったことを示します。
  </Step>
  <Step num="4" title="実行とフォロースルー">
    ボタンを離してスイングします。ホーミングショットの場合、ボールは自動的に最も近い有効なターゲット（エイムに応じてカップまたはプレイヤー）を追尾します。
  </Step>
</Steps>

| ショットスタイル | ロジック | 結果 |
| :--- | :--- | :--- |
| **パット・オーバー** | 高角度 / 低パワー | グリーン上で高い精度でスムーズに転がる |
| **チップ・アンダー** | 低角度 / 高パワー | 障害物や地雷を飛び越えるための高さを得る |
| **ホーミングショット** | エネルギー依存 | わずかな風を無視する高速追尾ショット |
| **マインパット** | 戦術的 | 地雷を友達のグループに打ち込んで混乱を引き起こす |

<Callout type="info" title="エキスパート戦略">
アイテムに対してもホーミングショットを実行できます！自分の前に地雷を落とし、ホーミングショットを使ってその地雷をトップのプレイヤーに直接飛ばすことができます。これは**Super Battle Golfの操作**の最も高度な使い方の1つです。
</Callout>

</Section>

<Section icon={Shield} title="地雷管理とハザードの解除">

地雷は最も頻繁に遭遇するハザードであり、他のプレイヤーによって設置されたり、マップ自体から出現したりします。起動中の地雷に踏み込むと吹き飛ばされ、そのラウンドを台無しにする可能性があります。生き残るためには、解除のための特定の**Super Battle Golfの操作**を学ぶことが不可欠です。

**地雷を安全に扱う方法:**
1. **状態の確認**: 起動した地雷には目に見える光やタイマーがあります。
2. **注意して接近**: 起動した地雷に向かって走らないでください。
3. **低速歩行**: **Altキー**（または指定された低速歩行ボタン）を長押しして、センサーを反応させずに接近します。
4. **解除**: 範囲内に入ったら**Eキー**を押して地雷を拾い、インベントリに追加します。

| 地雷アクション | 必要条件 | 結果 |
| :--- | :--- | :--- |
| **地雷を投げる** | アイテムスロット + クリック | 地雷を前方に発射。短い起動タイマーが開始 |
| **地雷を置く** | インベントリアクション | 足元にスタック全体を落とす。即座に起動 |
| **地雷を解除する** | Alt + E | 安全にハザードを取り除き、アイテムとして入手 |
| **ホーミングマイン** | 最大エネルギースイング | 静止した地雷を追尾弾に変える |

<Callout type="success" title="インベントリのコツ">
アイテムボックスの上に立った状態でラウンドを終了すると、動けない間もインベントリがいっぱいになるまでアイテムを拾い続けます。これを利用して次のホールのために備蓄しましょう！
</Callout>

</Section>
