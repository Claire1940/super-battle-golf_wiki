<Section icon={Monitor} title="パフォーマンスとグラフィック：Super Battle Golf 最良設定">

8人のプレイヤーが軌道レーザーやレールガンを同時に発射するゲームで勝利するには、ハードウェアがそれに対応できなければなりません。グラフィックに関する **Super Battle Golf 最良設定**は、視認性の最大化と安定した60 FPS以上の確保に焦点を当てています。スイングパワーのタイミングは視覚的なフィードバックに依存しているため、高いフレームレートは非常に重要です。スタッター（カクつき）が発生すると、カップを通り越したり、重要なチェックポイントを逃したりする原因になります。

| 設定カテゴリー | 推奨値 | パフォーマンスへの影響 |
| :--- | :--- | :--- |
| **ディスプレイモード** | フルスクリーン | 低（入力遅延を軽減） |
| **解像度** | ネイティブ (例: 1920x1080) | 高 |
| **フレームレート制限** | モニターのリフレッシュレートに合わせる | 中 |
| **影の品質** | 中 | 高 |
| **アンチエイリアス** | FXAA または SMAA | 中 |
| **モーションブラー** | 無効 | 低（鮮明さを向上） |
| **垂直同期 (V-Sync)** | オフ | 低（遅延を軽減） |

視野角（FOV）の調整も戦略的な選択です。FOVを広くすると、ライバルがクラブを振る様子や、こちらに向かってくる飛び道具など、側面からの攻撃に気づきやすくなります。しかし、FOVが広すぎると遠距離ショットの際にカップが小さく見え、狙いにくくなる可能性があります。

<Callout type="warning" title="ハードウェアに関する警告">
「Attack on City」アップデートのマップでフレームドロップが発生する場合は、**「エフェクト品質」**を下げてください。都市バイオームの車両や移動する障害物の密度の高さは、マルチプレイヤーでのアイテムの応酬中に古いGPUに負荷をかける可能性があります。
</Callout>

</Section>

適切な入力方法を選択することは、**Super Battle Golf 最良設定**を最適化するための第一歩です。ゲームは両方をネイティブにサポートしており、プレイスタイルに応じてそれぞれ明確な利点があります。

<CardGrid cols={2}>
  <Card title="キーボード＆マウス">
    - **精密な照準**: 遠距離のレールガンショットや軌道レーザーの配置に最適。
    - **カメラ操作**: 180度の振り向きが速く、背後の敵を確認しやすい。
    - **キー割り当て**: インベントリスロット1-5に素早くアクセスでき、アイテムを即座に使用可能。
  </Card>
  <Card title="コントローラー (PS5/Xbox)">
    - **アナログ移動**: ゴルフカートのステアリングがスムーズで、キャラクターの細かな移動が可能。
    - **トリガー感度**: スイングパワーの微調整において、より優れた触覚フィードバック。
    - **エルゴノミクス**: 全63ホールを巡る長時間のセッションに理想的。
  </Card>

キーボードを使用する場合、「ダイブ」と「アイテム使用」キーのリマッピングを強くお勧めします。デフォルトでは、移動用の指と干渉する可能性があります。「アイテム使用」をマウスのサイドボタンに移動させると、最終アプローチを隠すためにスモークボムを展開する際の反応速度が大幅に向上します。

| アクション | デフォルト（キーボード） | 推奨マッピング | コントローラー（デフォルト） |
| :--- | :--- | :--- | :--- |
| **スイング/ショット** | 左クリック | 左クリック | 右トリガー |
| **ジャンプ** | スペース | スペース | A / Cross |
| **ダイブ** | Shift | マウス 4 または Q | B / Circle |
| **アイテム使用** | E | マウス 5 または F | 右バンパー |
| **アイテムを捨てる** | G | X | 方向キー下 |
| **カートに乗る** | F | E | X / Square |

<Callout type="tip" title="プロの操作ヒント">
設定メニューで「スプリント切り替え」を有効にしましょう。これにより、Shiftキーを小指で押し続ける必要がなくなり、海岸や雪原バイオームの隙間を飛び越えるために必要なジャンプ・ダイブのコンボ移動に完全に集中できるようになります。
</Callout>

<Section icon={Target} title="ゲームプレイとスイング感度の設定">

ゲームの核心はスイングのメカニズムです。感度に関する **Super Battle Golf 最良設定**を見つけることは、ホールインワンと崖からの転落を分ける決定的な要素となります。感度は、照準レティクルの移動速度や、ショット間の混沌とした移動フェーズ中にカメラをどれだけ速く調整できるかに影響します。

<Steps>
  <Step num="1" title="照準感度の調整">
    まずは中程度の感度から始めてください。パッティング中に修正しすぎてしまう場合は、水平感度を下げてください。移動するプレイヤーを追跡できる速さと、狭いグリーンを狙える精度の両立が必要です。
  </Step>
  <Step num="2" title="カメラ追従速度の調整">
    設定の「カメラ追従」を確認してください。これを「高」に設定すると、ボールを打った後すぐにカメラがキャラクターに追従し、即座に次のチェックポイントへ走り出すことができます。
  </Step>
  <Step num="3" title="スイングパワーHUDの微調整">
    パワーメーターを「高視認性」モードに設定してください。「Frozen Fairway」のようなマップでは、眩しい雪によってデフォルトのUIカラーが見えにくくなることがあります。
  </Step>
</Steps>

| 感度ティア | 値 (0-100) | 最適な用途 |
| :--- | :--- | :--- |
| **低** | 15-30 | 精密なパッティングと遠距離狙撃。 |
| **バランス** | 45-60 | 一般的なプレイ。初心者にお勧め。 |
| **高** | 75-90 | 攻撃的な戦闘と迅速なナビゲーション。 |

<Callout type="success" title="精度アップ">
特に「垂直感度」を下げることで、照準を水平に保ちやすくなり、平坦なフェアウェイで誤って地面や空を狙うことなく、真っ直ぐなドライブを打ちやすくなります。
</Callout>

</Section>

<Section icon={Swords} title="戦闘とアイテム管理の設定">

Super Battle Golf では、アイテムはクラブと同じくらい重要です。インベントリを効果的に管理するには、UIとクイックアクセスに関する特定の **Super Battle Golf 最良設定**が必要です。2026年7月のアップデートでは、フラッシュカメラやレールガンなど、正確なタイミングを必要とする新アイテムがいくつか導入されました。

**最適なアイテム使用戦略：**
- **レールガンの精度**: マウスのDPIまたは感度設定を低くして、移動するターゲットを偏差射撃しやすくします。
- **スモークボムのタイミング**: 自分の位置に「軌道レーザー」の警告が表示されたら展開し、正確な移動経路を隠します。
- **サンダーストームの配置**: マップオーバーレイ（Tabキー）を使用して、プレイヤーが密集している狭い橋やチェックポイントに範囲攻撃を配置します。

| アイテム名 | カテゴリー | 最適な使用場面 |
| :--- | :--- | :--- |
| **レールガン** | 攻撃 | フェアウェイの向こう側にいるリーダーを狙撃する。 |
| **軌道レーザー** | エリア制御 | 最終グリーンへのアクセスをブロックする。 |
| **ジャンボバーガー** | 防御 | カップ付近の混雑したクラブ戦で生き残る。 |
| **フラッシュカメラ** | 妨害 | 相手の最終パット中に視界を奪う。 |

<Callout type="tip" title="インベントリ管理">
ゲームプレイ設定で「弱いアイテムを自動で捨てる」を有効にしてください。これにより、レールガンのようなSティアアイテムの上を歩いた際に、低ティアのアイテムが自動的にドロップされるため、手動でインベントリを整理する貴重な時間を節約できます。
</Callout>
