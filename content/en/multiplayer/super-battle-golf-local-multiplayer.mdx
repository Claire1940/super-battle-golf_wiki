<Section icon={Settings} title="Advanced Setup: OBS 4-Screen Split Configuration">

As highlighted in recent community setups, using OBS Studio to manage your **super battle golf local multiplayer** layout can provide a much cleaner interface than the default in-game options. This is particularly useful for streamers or those using multiple monitors.

<Steps>
  <Step num="1" title="Launch Multiple Instances">
    Open the game in windowed mode. If the game supports multiple local profiles, ensure each is assigned to a specific controller via the Steam Input settings.
  </Step>
  <Step num="2" title="Source Capture in OBS">
    Create a new Scene in OBS. Add a 'Window Capture' source for each player's view. If the game uses a single window for split-screen, you will capture the same window multiple times.
  </Step>
  <Step num="3" title="Apply Cropping Filters">
    Right-click on a source, go to 'Filters', and add a 'Crop/Pad' filter. Adjust the margins so that only Player 1's quadrant is visible for the first source, Player 2's for the second, and so on.
  </Step>
  <Step num="4" title="Layout Arrangement">
    Resize and snap each cropped source into a 2x2 grid. This allows you to add custom overlays, player names, and facecams between the gameplay windows without obscuring the action.
  </Step>
</Steps>

<Callout type="success" title="Streamer Advantage">
Using the OBS method allows you to output a "clean" feed to your friends' monitors while keeping a "detailed" feed with stats and chat on your primary screen.
</Callout>

This setup is the preferred method for local tournaments. It ensures that every player has a consistent UI scale, which can sometimes get distorted in native ultra-wide split-screen modes.

</Section>
