<Section icon={Target} title="Step-by-Step OBS Split Screen Setup">

To achieve a functional **super battle golf split screen** experience, you must follow a specific technical workflow. This involves launching separate instances of the game and using Open Broadcaster Software (OBS) to composite them into a single view. This method is ideal for those using a large TV or a projector for local gatherings.

<Steps>
  <Step num="1" title="Prepare Multiple Game Instances">
    Ensure your PC can launch the game multiple times. You may need to use separate Steam accounts or specific launch parameters (like `-multi-instance`) depending on your current version. Each instance must be set to Windowed Mode in the graphics settings.
  </Step>
  <Step num="2" title="Configure Controller Mapping">
    Connect all controllers before launching the instances. Use a tool like DS4Windows or Steam Input to ensure Instance 1 only listens to Controller A, and Instance 2 only listens to Controller B. This prevents one controller from moving all characters at once.
  </Step>
  <Step num="3" title="OBS Scene Composition">
    Open OBS and create a new scene. Add each game window as a separate "Window Capture" source. Right-click each source, go to Transform, and resize them to fit into quadrants (e.g., top-left, top-right) for a 4-player split.
  </Step>
  <Step num="4" title="Audio Management">
    To avoid an echo chamber, mute the audio on all instances except for one. This ensures you get the game music and sound effects without the overlapping noise of four identical audio streams.
  </Step>
</Steps>

<Callout type="success" title="Pro Setup Tip">
If you have a multi-monitor setup, you can extend the OBS Projector to your secondary screen (like a 4K TV) while keeping your primary monitor free for technical adjustments during the match.
</Callout>
