<Section icon={Map} title="Экспорт карт из Unity">

Процесс экспорта — это самая техническая часть создания **пользовательских карт Super Battle Golf**. Вам необходимо различать пакеты сцен (scene bundles) и пакеты ассетов (asset bundles). Пакет сцены содержит фактический макет уровня (лунку), в то время как пакет ассетов содержит пользовательские 3D-модели или текстуры, которые вы импортировали в проект.

**Основные моменты видео:**
- **Именование пакетов**: как использовать соглашение об именах `username.packname`.
- **Консоль сборки**: использование команды `B:` для проверки содержимого пакета.
- **Конвейер экспорта**: навигация по меню сборки для генерации финальных файлов.
- **Многосценовые паки**: добавление нескольких лунок в один пакет ассетов.

<Callout type="warning" title="Ограничение пакетов ассетов">
Вы не можете смешивать сцены и другие ассеты (например, текстуры или модели) в одном пакете. Если вы используете пользовательские объекты, которых нет в базовой игре, вы должны создать отдельный пакет (например, `username.assets`) и сослаться на него соответствующим образом.
</Callout>

<Steps>
  <Step num="1" title="Очистка пакетов ассетов">
    Перед сборкой перейдите в меню Build в Unity и выберите «Clear All Asset Bundle Names». Это гарантирует, что старые данные не помешают вашему новому экспорту.
  </Step>
  <Step num="2" title="Назначение имен пакетов">
    Выберите файл сцены в окне Project. В инспекторе (внизу справа) создайте новое имя Asset Bundle. Используйте формат `ВашеИмя.ИмяКарты`.
  </Step>
  <Step num="3" title="Сборка пакетов">
    Перейдите в меню «Build» и выберите «Build Asset Bundles». Этот процесс может занять 1–2 минуты на сцену в зависимости от вашего оборудования.
  </Step>
  <Step num="4" title="Поиск экспортированных файлов">
    Откройте папку проекта в проводнике Windows и перейдите в `Assets/Bundles`. Вам нужен только файл без расширения (игнорируйте файлы `.manifest`).
  </Step>
</Steps>

</Section>

<Section icon={Target} title="Настройка метаданных JSON">

JSON-файл служит «удостоверением личности» для ваших **пользовательских карт Super Battle Golf**. Без правильно настроенного JSON игра не будет знать, как отображать вашу карту в меню или каковы правила подсчета очков. Вы можете скопировать `example.json`, предоставленный в папке мода, и отредактировать его в любом текстовом редакторе, например Notepad++.

| Ключ JSON | Тип значения | Описание |
| :--- | :--- | :--- |
| **sceneName** | Строка | Должно точно соответствовать имени сцены в Unity |
| **holeName** | Строка | Имя, отображаемое игрокам в меню |
| **par** | Целое число | Стандартное количество ударов для лунки |
| **difficulty** | Строка | Easy, Medium, Hard или Expert |
| **enabled** | Логическое | Должно быть установлено в `true`, чтобы отображаться в игре |

<Callout type="success" title="Поддержка нескольких лунок">
Чтобы добавить несколько лунок в один пак, просто поставьте запятую после закрывающей фигурной скобки первой записи лунки и вставьте новый набор фигурных скобок с данными второй лунки.
</Callout>

</Section>
